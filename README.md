# CustomHealthIndicator

A Minecraft Fabric 1.16.5 client-side mod that shows enemy health in real time.

## Features

- **Colored nametags** — player nametags display current HP with a heart icon. Color shifts from green to red as HP drops, and pulses when critically low
- **Flying damage numbers** — when you hit someone, a damage number flies out from your crosshair in a random direction
- **Big HP number** — a large red number appears below your crosshair showing target's current HP, disappears after 2 seconds

## Installation

1. Download `customhealthindicator-1.5-fabric-1.16.5.jar` from [Releases](../../releases)
2. Place it in your `.minecraft/mods/` folder
3. Launch Minecraft with **Fabric Loader 1.16.5**

No Fabric API required.

## Requirements

- Minecraft 1.16.5
- Fabric Loader 0.13+
